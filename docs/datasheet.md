# Parts spec
- ESP32-WROOM-32
    - Chip: ESP32-D0WDQ6 (dual-core)
    - Features: WiFi + Bluetooth Classic + BLE
    - Flash: 4MB (most common)
    - RAM: 520KB SRAM
    - Cores: 2 cores @ 240MHz
